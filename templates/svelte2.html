Svelte 2.0

Svelte 2.0 is an online programme conducting by TinkerHub GECT, it's an introduction to Basics of Flask in Python. Flask is a beginner-friendly web framework that allows you to build web applications using Python.In this programme, we will cover key concepts in Flask, including connecting to a database, routing with variable rules, understanding templates, using Jinja2 for templating, handling HTTP methods (GET/POST), and retrieving form data.
